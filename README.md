# echo

这是我的 Python Web 课程设计，目前仍在开发。

轻量级极客音频流媒体社区的 Django 项目目录。

本项目**大量**使用AI生成代码

## 在 PyCharm 中打开

直接打开当前目录：

```text
D:\Code\PythonProjects\echo
```

解释器选择项目内虚拟环境：

```text
D:\Code\PythonProjects\echo\.venv\Scripts\python.exe
```

如果 PyCharm 仍提示没有 Django，重新选择一次上面的解释器并等待索引完成。

## 启动页面

```bat
tools\run_dev_server.bat
```

访问：

```text
http://127.0.0.1:8000/
```
